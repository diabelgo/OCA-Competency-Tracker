📲 Add to Home Screen

Follow the steps for your device to install the tracker as an app.

📱 Android (Chrome)
1. Tap the three-dot menu ⋮ (top-right corner)
2. Tap "Add to Home screen"
3. Tap Add — done ✓
📱 iPhone / iPad (Safari)
1. Tap the Share button ⬆ (bottom centre)
2. Scroll down and tap "Add to Home Screen"
3. Tap Add — done ✓
Must use Safari — Chrome on iPhone cannot install apps
💻 Windows
Chrome (easiest):
1. Click ⋮ menu (top-right)
2. Click "Create shortcut..."
3. Tick "Open as window" → click Create
A shortcut appears on your Desktop ✓

Edge:
1. Click … menu (top-right)
2. Click Apps → Install this site as an app
3. Click Install ✓
💡 "Install page as app" in Chrome only appears when the tracker is on a web server (https://). "Create shortcut" always works from a local folder.
💻 Mac (Chrome)
Click ⋮ menu → Save and share → Install page as app
Find it in Applications and drag to your Dock ✓
💡 Tip: After adding to Home Screen, the tracker opens full-screen. Your data saves on your device — no internet needed.
